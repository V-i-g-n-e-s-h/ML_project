# ML_project

## clone repo:
```
git clone <repo-url>
cd ML_project
```

## Create Virtual environment:
```
python -m venv venv
```
### On Ubuntu
```
source venv/bin/activate  
```
### On Windows: 
```
venv\Scripts\activate
```

## Install the Required Packages
```
pip install -r requirements.txt
```

## Execute the code
```
streamlit run app.py
```
